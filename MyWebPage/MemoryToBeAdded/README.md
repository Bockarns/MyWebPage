# Individuellet Projekt 
Detta är ett memoryspel som gjordes som ett individuellt programmeringsprojekt i JavaScript 1, med följande funktioner: <br><br>

*Struktur* <br>
-HTML<br>
-CSS<br>
-JS<br><br>
*Funktioner*<br>
-Välj tid<br>
-Starta<br>
-Börja om<br>
-Flippräknare<br>
-Modal för vinst/förlust med information om tid, antal par hittade/kvar och antal försök<br>
-Möjlighet att byta kort i JS kod (möjlighet till utveckling för att randomisera flera kort. Eller välja kategori av kort.)<br>
